📘 Nexum Networking Documentation

    Overview


    The CODEC library of the Networking system.
    Handles data serialization, compression, encryption, and Base64 encoding/decoding for secure network communication.
    Supports JSON, ChaCha20, Poly1305, LZW, and Base64 libraries for full data protection and compression workflow.

    LIBRARY:Initialize


    Initializes the CODEC instance with provided libraries and an encryption key.
    Validates that each dependency library exposes required functions.
    The encryption key is processed to support hex escape sequences.
    Returns a CODEC instance with JSON, CHACHA20, POLY1305, LZW, BASE64, and KEY fields.

    Parameters

    • table : JSON library with encode and decode functions
    • table : ChaCha20 library with encrypt and decrypt functions
    • table : Poly1305 library with auth function
    • table : LZW library with compress and decompress functions
    • table : Base64 library with encode and decode functions
    • string : Encryption key string

    Returns

    • table : CODEC instance table
    local Nexum         = require("srcs")  
local tInstance     = Nexum:Instanciate("networking", "SUPER-CLIENT")  
local sLibraryName  = "CODEC"  
local tLibrary      = tInstance:GetLibrary(sLibraryName)

    LIBRARY:IsValidData


    Checks if a data table conforms to CODEC format.
    Valid data must have an ID string, a CONTENT field, and boolean ENCRYPTED and COMPRESSED flags.

    Parameters

    • table : Data table to validate

    Returns

    • boolean : true if valid, false otherwise
    local Nexum         = require("srcs")  
local tInstance     = Nexum:Instanciate("networking", "SUPER-CLIENT")  
local sLibraryName  = "CODEC"  
local tLibrary      = tInstance:GetLibrary(sLibraryName)  
local bValid        = tLibrary:IsValidData({ID="MSG", CONTENT="Hello", ENCRYPTED=false, COMPRESSED=true})

    LIBRARY:Compress


    Compresses a string using the LZW library.
    Returns the compressed string or prints an error on failure.

    Parameters

    • string : Content string to compress

    Returns

    • string : Compressed content
    local Nexum         = require("srcs")  
local tInstance     = Nexum:Instanciate("networking", "SUPER-CLIENT")  
local sLibraryName  = "CODEC"  
local tLibrary      = tInstance:GetLibrary(sLibraryName)  
local sCompressed   = tLibrary:Compress("Some text data to compress")

    LIBRARY:Decompress


    Decompresses a string using the LZW library.
    Returns the original content or prints an error on failure.

    Parameters

    • string : Compressed content string

    Returns

    • string : Decompressed content
    local Nexum         = require("srcs")  
local tInstance     = Nexum:Instanciate("networking", "SUPER-CLIENT")  
local sLibraryName  = "CODEC"  
local tLibrary      = tInstance:GetLibrary(sLibraryName)  
local sContent      = tLibrary:Decompress(sCompressed)

    LIBRARY:Encrypt


    Encrypts a string with ChaCha20 and generates a Poly1305 authentication tag.
    Requires a key and a nonce.

    Parameters

    • string : Content string to encrypt
    • string : Encryption key
    • string : Nonce string

    Returns

    • string : Encrypted content
    • string : Poly1305 tag
    local Nexum         = require("srcs")  
local tInstance     = Nexum:Instanciate("networking", "SUPER-CLIENT")  
local sLibraryName  = "CODEC"  
local tLibrary      = tInstance:GetLibrary(sLibraryName)  
local sEncrypted, sTag = tLibrary:Encrypt("Hello World", "myKey", "123456789012")

    LIBRARY:Decrypt


    Decrypts content using ChaCha20 and verifies integrity with Poly1305 tag.
    Returns the decrypted string or prints an error if the tag is invalid or decryption fails.

    Parameters

    • string : Encrypted content
    • string : Encryption key
    • string : Nonce string
    • string : Poly1305 tag

    Returns

    • string : Decrypted content
    local Nexum         = require("srcs")  
local tInstance     = Nexum:Instanciate("networking", "SUPER-CLIENT")  
local sLibraryName  = "CODEC"  
local tLibrary      = tInstance:GetLibrary(sLibraryName)  
local sDecrypted    = tLibrary:Decrypt(sEncrypted, "myKey", "123456789012", sTag)

    LIBRARY:Base64Encode


    Encodes content, tag, and nonce to Base64.
    Returns Base64-encoded versions or prints errors if encoding fails.

    Parameters

    • string : Content string
    • string : Poly1305 tag
    • string : Nonce string

    Returns

    • string : Base64-encoded content
    • string : Base64-encoded tag
    • string : Base64-encoded nonce
    local Nexum         = require("srcs")  
local tInstance     = Nexum:Instanciate("networking", "SUPER-CLIENT")  
local sLibraryName  = "CODEC"  
local tLibrary      = tInstance:GetLibrary(sLibraryName)  
local sB64Content, sB64Tag, sB64Nonce = tLibrary:Base64Encode(sEncrypted, sTag, "123456789012")

    LIBRARY:Base64Decode


    Decodes Base64-encoded content, tag, and nonce.
    Returns decoded versions or prints errors if decoding fails.

    Parameters

    • string : Base64-encoded content
    • string : Base64-encoded tag
    • string : Base64-encoded nonce

    Returns

    • string : Decoded content
    • string : Decoded tag
    • string : Decoded nonce
    local Nexum         = require("srcs")  
local tInstance     = Nexum:Instanciate("networking", "SUPER-CLIENT")  
local sLibraryName  = "CODEC"  
local tLibrary      = tInstance:GetLibrary(sLibraryName)  
local sContent, sTag, sNonce = tLibrary:Base64Decode(sB64Content, sB64Tag, sB64Nonce)

    LIBRARY:Encode


    Encodes a data table for network transmission.
    Applies JSON encoding, optional compression, optional encryption with Poly1305, and Base64 encoding.

    Parameters

    • table : Data table with fields ID, CONTENT, ENCRYPTED, COMPRESSED

    Returns

    • string : JSON string ready to send
    local Nexum         = require("srcs")  
local tInstance     = Nexum:Instanciate("networking", "SUPER-CLIENT")  
local sLibraryName  = "CODEC"  
local tLibrary      = tInstance:GetLibrary(sLibraryName)  
local sData         = tLibrary:Encode({ID="MSG_HELLO", CONTENT={text="Hi"}, ENCRYPTED=true, COMPRESSED=true})

    LIBRARY:Decode


    Decodes a JSON string received from the network.
    Applies Base64 decoding, optional decryption, optional decompression, and JSON decoding.
    Returns the ID and original content table.

    Parameters

    • string : JSON string received

    Returns

    • string : Message ID
    • table : Original content table
    local Nexum         = require("srcs")  
local tInstance     = Nexum:Instanciate("networking", "SUPER-CLIENT")  
local sLibraryName  = "CODEC"  
local tLibrary      = tInstance:GetLibrary(sLibraryName)  
local sID, tContent = tLibrary:Decode(sData)
