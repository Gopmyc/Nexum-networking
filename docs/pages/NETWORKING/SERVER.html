📘 Nexum Networking Documentation

    Overview


    The Networking server class of Nexum.
    Manages client-server communication, event handling, hooks, message IDs,
    encryption and compression of packets, and integration with the Nexum runtime.
    It allows configuration-based defaults and per-packet overrides for secure and optimized communication.

    CLASS:Initialize


    Initializes the Networking server instance.
    Validates ENet library dependency and required network configuration parameters
    (IP, PORT, MAX_CLIENTS, MAX_CHANNELS, bandwidth, timeout, encryption/compression defaults).
    Creates the ENet host, initializes HOOKS, CODEC, and EVENTS libraries.
    Returns a server instance table containing all network state and handlers.

    Returns

    • table : Server instance containing HOST, CLIENTS, NETWORK_ID, HOOKS, CODEC, EVENTS, DEFAULT_ENCRYPT, DEFAULT_COMPRESS
    local Nexum         = require("srcs")
local tInstance     = Nexum:Instanciate("networking", "SUPER-SERVER")

    CLASS:GetHost


    Returns the underlying ENet host object used for server communication.

    Returns

    • userdata : ENet host
    local Nexum         = require("srcs")
local tInstance     = Nexum:Instanciate("networking", "SUPER-SERVER")
local tHost			= tInstance:GetHost()

    CLASS:Update


    Processes incoming network events and dispatches them through the EVENTS system.
    Each event is safely executed with xpcall to catch errors.

    Parameters

    • number : Optional delta time (iDt)
    local Nexum         = require("srcs")
local tInstance     = Nexum:Instanciate("networking", "SUPER-SERVER")
tInstance:Update(dt)

    CLASS:SendToClient


    Sends a packet to a single client.
    Validates client ID and packet table, applies default or provided flags for reliability.
    Triggers the 'send' event through EVENTS.

    Parameters

    • string : Client ID
    • table : Packet data
    • number : Optional channel ID (default 0)
    • string : Optional flag ("reliable", "unreliable", "unsequenced")
    local Nexum         = require("srcs")
local tInstance     = Nexum:Instanciate("networking", "SUPER-SERVER")
local tPacket		= tInstance:BuildPacket("MSG_HELLO", {text="Hello"}, true, false)
tInstance:SendToClient("client_01", packet, 0, "reliable")

    CLASS:SendToClients


    Sends the same packet to all currently connected clients.
    Skips invalid or empty client entries.

    Parameters

    • table : Packet data
    • number : Optional channel ID
    • string : Optional flag
    local Nexum         = require("srcs")
local tInstance     = Nexum:Instanciate("networking", "SUPER-SERVER")
tInstance:SendToClients(packet)

    CLASS:BuildPacket


    Creates a network packet table with optional encryption and compression.
    If flags are not provided, defaults from server configuration are applied.

    Parameters

    • string : Message ID
    • var : Packet content
    • boolean : Optional encrypt flag
    • boolean : Optional compress flag

    Returns

    • table : Packet containing ID, CONTENT, ENCRYPTED, COMPRESSED
    local Nexum         = require("srcs")
local tInstance     = Nexum:Instanciate("networking", "SUPER-SERVER")
local tPacket 		= tInstance:BuildPacket("MSG_UPDATE", {posX=10, posY=20}, true, true)

    CLASS:AddNetworkID


    Adds a message ID to the server's valid network IDs list.

    Parameters

    • string : Message ID
    local Nexum         = require("srcs")
local tInstance     = Nexum:Instanciate("networking", "SUPER-SERVER")
tInstance:AddNetworkID("MSG_UPDATE")

    CLASS:SubNetworkID


    Removes a message ID from the server's valid network IDs list.

    Parameters

    • string : Message ID
    local Nexum         = require("srcs")
local tInstance     = Nexum:Instanciate("networking", "SUPER-SERVER")
tInstance:SubNetworkID("MSG_UPDATE")

    CLASS:IsValidClient


    Checks if a client ID exists and is currently registered on the server.

    Parameters

    • string : Client ID

    Returns

    • table : Client data table if valid
    • boolean : false if client is invalid
    local Nexum         = require("srcs")
local tInstance     = Nexum:Instanciate("networking", "SUPER-SERVER")
local clientData 	= tInstance:IsValidClient("client_01")
if clientData then
	print("Client is valid")
end

    CLASS:IsValidMessage


    Checks if a message ID is registered as valid for this server.

    Parameters

    • string : Message ID

    Returns

    • boolean : true if message ID is valid
    local Nexum         = require("srcs")
local tInstance     = Nexum:Instanciate("networking", "SUPER-SERVER")
if tInstance:IsValidMessage("MSG_HELLO") then
	print("Message is allowed")
end

    CLASS:AddHook


    Registers a callback hook for network events.
    Hooks can be executed on connect, disconnect, send, receive, or unhandled events.

    Parameters

    • string : Hook ID
    • function : Callback function
    local Nexum         = require("srcs")
local tInstance     = Nexum:Instanciate("networking", "SUPER-SERVER")
tInstance:AddHook("onReceive", function(client, data)
	print("Received data from", client, data)
end)

    CLASS:Destroy


    Cleans up the networking server instance.
    Kicks all clients, clears CLIENTS and NETWORK_ID tables, destroys HOOKS and EVENTS libraries,
    flushes the ENet host, and removes metatable references to fully release resources.
